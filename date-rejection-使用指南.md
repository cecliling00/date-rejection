# date-rejection 约会拒绝话术 Skill — 使用指南

## 一句话概述

当你收到约会邀请不知道怎么拒绝时，告诉 WorkBuddy 对方说了什么，Skill 自动生成一句荒诞但逻辑自洽的拒绝话术。

---

## 1. 怎么触发这个 Skill

Skill 已经安装到你本地的 `~/.workbuddy/skills/date-rejection/` 目录。只要你在 WorkBuddy 中提到以下任意关键词，Skill 就会自动激活：

| 触发关键词 | 示例说法 |
|-----------|---------|
| 约会拒绝 | "给我一个约会拒绝话术" |
| 拒绝话术 | "帮我想个拒绝话术" |
| 怎么拒绝 | "TA 约我吃饭，怎么拒绝" |
| 不想去 | "有人约我看电影，不想去怎么回" |
| 约我 | "TA 约我周末出去" |
| 怎么回 | "他约我看《流浪地球》，怎么回" |
| 拒绝约会 | "帮我拒绝约会邀请" |
| 拒绝邀请 | "给我一句拒绝邀请的话" |
| 看电影拒绝 / 约饭拒绝 | "约饭拒绝理由" |

**不需要说「请调用 date-rejection skill」这种话**，自然聊到约会拒绝就会自动触发。

---

## 2. 三种使用方式

### 方式 A：给出邀请原话（推荐）

直接把对方发的消息贴过来，Skill 会提取关键信息生成定制化拒绝。

> **你**：TA 说「周末一起去看《封神》吧」
>
> **WorkBuddy**：这个电影的时长太短了，不适合在电影院看 —（活动属性荒诞化）

### 方式 B：描述邀请场景

用自己的话描述对方约你做什么，Skill 也能识别。

> **你**：同事约我下班后去新开的那家川菜馆吃饭，不想去
>
> **WorkBuddy**：这家餐厅评分太高了，不适合工作日去吃 —（活动属性荒诞化）

### 方式 C：指定拒绝模式

如果你想要特定风格的拒绝，在场景描述后加上模式要求。

> **你**：朋友约我周末爬山，用神秘时间窗口模式帮我拒绝
>
> **WorkBuddy**：我能出门的时间段有限，只有上午 9:14 到 9:47 这个窗口，你的爬山在这个窗口之外 —（神秘时间窗口）

## 3. 六种模式使用场景速查

| 模式 | 最适合的拒绝场景 | 拒绝风格 |
|------|-----------------|---------|
| 活动属性荒诞化 | 看电影、去餐厅、参加活动 | "这个 X 太 Y 了，不适合 Z" |
| 荒诞优先级 | 约饭、约会、吃饭 | "我今天要 X，等会还要 X，不 Y 了" |
| 环境外部归因 | 出门、户外活动 | "今天 X 太 Y 了，不适合出门" |
| 神秘时间窗口 | 所有类型 | "我能出门的时间段有限，只有 X" |
| 一次性体验论 | 去特定地点 | "这里我去过了，X" |
| 身心状态哲学化 | 所有类型 | "我现在处于 X 阶段，不适合 Y" |

## 5. 特殊情况处理

### 你没说对方说了什么

如果只说「帮我拒绝」而没给邀请内容，WorkBuddy 会追问：

> **你**：帮我拒绝约会
>
> **WorkBuddy**：对方具体是怎么约你的？把邀请内容发给我，我来生成拒绝话术。

### 复杂邀请（比如又吃饭又看电影）

Skill 会自动拆分成两个邀请分别拒绝：

> **你**：TA 约我先吃饭再看电影
>
> **WorkBuddy**：
> - 吃饭：我今天要思考中午吃什么，思考完还要思考晚上吃什么，太忙了 —（荒诞优先级）
> - 看电影：这个电影时长太长了，不适合在电影院看 —（活动属性荒诞化）

## 6. 效果进阶：组合模式

当你想要更高级的拒绝，可以说「用组合模式」或直接指定两种模式：

> **你**：TA 约我去三里屯那家新开的火锅店，用环境归因+一次性体验组合
>
> **WorkBuddy**：今天湿度太高不适合吃火锅，而且三里屯我已经去过了，一个人的区域配额只能去一次 —（环境外部归因 + 一次性体验论）

---

## 7. 快速上手三句话

1. **直接把对方约你说了什么告诉 WorkBuddy** —— 越具体越好
2. **不指定模式则自动匹配** —— Skill 会根据邀请类型选最合适的
3. **想控制风格就加一句**：「用 XX 模式」即可

---

## 8. 温馨提示

- 话术仅供娱乐，真实社交请自行判断时机和分寸
- 拒绝话术每次生成不重复，因为会结合你输入的具体信息
- Skill 安装位置：`~/.workbuddy/skills/date-rejection/`
- 如需调整触发词或添加新模式，可直接修改 `SKILL.md` 和 `references/rejection_patterns.md`
